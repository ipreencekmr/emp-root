# emp-root

Employee Management Root module is dependent on following child modules

- emp-header
- emp-footer
- emp-container
- emp-home
- emp-employee
- emp-address


## Tech Stack Used

- React
- JavaScript
- Bootstrap
- Micro Front End
- Jest
- React Testing Library
- ESLint
- Lighthouse
- A11y
- Docker 
- Docker Compose
- AWS ECS
- Github Actions

## Run 

`docker-compose up`
